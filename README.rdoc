== READ ME 

Ứng dụng hoàn chỉnh đầu tiên viết bằng Rails và AngularJS

Ứng dụng dùng để quản lý Kệ Sách nhà mình, dạng Single Page App, với các chức năng cơ bản như 
- CRUD
- Book Review with Markdown Editor
- Book Rating 
- Upload and Showing Book Cover 
- Authors Tag

> Hoàn toàn bằng AngularJS trên Front-End ( chiếm 70% ) và Rails - Back-End
